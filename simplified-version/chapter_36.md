# 第 36 章：将欲取之，必固与之

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
将欲歙（xī）之，必固张之；将欲弱之，必固强之；
将欲废之，必固兴之；将欲夺之，必固与之，是谓微明。
柔弱胜刚强。鱼不可脱于渊，国之利器不可以示人。
```

## 解释
**将欲歙（xī）之，必固张之；将欲弱之，必固强之；**
想要收缩它，必先暂且扩张它；想要削弱它，必先暂且增强它。

**将欲废之，必固兴之；将欲夺之，必固与之，是谓微明。**
想要废除它，必先暂且兴举它；想要夺取它，必先暂且给予它。这就是微妙的高明（洞察了事物向反面转化的规律）。

**柔弱胜刚强。鱼不可脱于渊，国之利器不可以示人。**
柔弱最终能战胜刚强。鱼儿不能离开赖以生存的深潭，国家的坚强力量（或权柄）也不可以轻易炫耀示人。

## 分析
本章展示了老子极其深邃的战略眼光。他发现事物在达到顶峰或极端后，必然会向相反的方向转化。这种“欲擒故纵”的智慧，不是为了阴谋诡计，而是为了顺应事物发展的自然节奏。当一个事物被过度扩张或增强时，其内部的矛盾也会随之激化，最终导致自我的崩塌。

在商业竞争或职业规划中，这种“微明”的智慧极具参考价值。当你面对一个极其强大的对手时，硬碰硬往往是下策；相反，你可以观察其是否正处于过度膨胀、利令智昏的阶段。老子告诉我们，真正的力量（利器）是深藏不露的。那些整天炫耀武力、急于扩张的人，往往已经离开了“道”的深渊，变得极度脆弱。

我们要学会“守弱”和“深藏”。不要在时机未成熟时急于表现自己的锋芒，也不要因为暂时的获取而沾沾自喜。保持在自己的“深渊”（核心领域和定力）中，默默打磨实力。记住，那些看起来柔弱、低调的东西，往往具有最持久的生命力；而那些过度张扬的东西，往往离凋谢已经不远了。
