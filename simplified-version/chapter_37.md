# 第 37 章：道常无位而无不为

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
道常无为而无不为，侯王若能守之，万物将自化。
化而欲作，吾将镇之以无名之朴。
无名之朴，夫亦将无欲。不欲以静，天下将自定。
```

## 解释
**道常无为而无不为，侯王若能守之，万物将自化。**
“道”永远是顺应自然而不妄为的，但它却通过这种方式成就了万物（无所不为）。君王如果能坚守这种原则，万事万物就会自然而然地生长发育。

**化而欲作，吾将镇之以无名之朴。**
在万物化育的过程中，如果产生了贪婪的欲望，我就用“道”的质朴本真（无名之朴）来引导它回归。

**无名之朴，夫亦将无欲。不欲以静，天下将自定。**
用这种质朴本真来引导，万物就不会产生过度的私欲。没有了私欲就能保持清静，天下自然也就会达到和谐安定的状态。

## 分析
本章是老子“无为而治”政治理想的总纲。他认为，世界本身具有自我调节、自我化育的能力（自化），管理者的任务不是去制造秩序，而是去守护秩序。当社会出现贪欲和混乱时，解决办法不是增加更多的禁令，而是通过引导人们回归“质朴”的初心，从根源上平息动乱。

在现代管理中，我们经常陷入“为了管理而管理”的怪圈：制定密集的规章、实施高压的监控。结果往往是员工失去了活力，甚至产生更多投机取巧的行为（欲作）。老子告诉我们，一个优秀的系统应该是具备自组织能力的。管理者要做的，是营造一种“无欲以静”的文化氛围，让每个人都能在自己的轨道上顺畅运作。

我们要学会在生活中寻求“定力”。当你的心境变得杂乱、欲望变得膨胀时，试着找回那份“无名之朴”——那些最简单、最本质的需求和原则。不要被外在的浮华所诱惑，保持内心的清静。当你能守住自己的本真时，你周围的小环境也自然会变得有序而和谐。大道至简，生活亦然。
