# 第 6 章：谷神不死，是谓玄牝

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
谷神不死，是谓玄牝（pìn）。
玄牝之门，是谓天地根。
绵绵若存，用之不勤。
```

## 解释
**谷神不死，是谓玄牝。**
“道”就像山谷里虚空而又充满神性的主宰（谷神）一样，是永恒长存的，这就是玄妙的母体。

**玄牝之门，是谓天地根。**
这种玄妙母体的生化产门，就是天地万物赖以生成的根源。

**绵绵若存，用之不勤。**
它像丝绵一样连续不断地存在着啊，作用无穷无尽，且永不枯竭。

## 分析
本章以极具诗意且形象的语言，界定了“道”的母性特质——生养与包容。老子将“道”比作空灵的山谷（谷神）与伟大的产门（玄牝），揭示了虚空才能承载万物，柔韧才能孕育生机的底层原理。这一逻辑强调了生命与创造的源泉，往往来自于宁静与深邃的“无”。

在现代生活中，这种“母性智慧”对我们的启发很大。很多时候，我们推崇的是阳刚、显露、直接的竞争（牡），而忽视了阴柔、含蓄、包容的承载力（牝）。无论是个人的成长还是团队的协作，若没有一个像山谷那样虚怀若谷的内心（谷神），便无法吸纳新的知识与能量。这种“绵绵若存”的力量，虽然看似微弱、不露锋芒，实则是最持久的动力。

行动上，我们要学会效法“玄牝”的智慧。在处理问题时，不一定要用强硬的干预（勤），而可以用顺应自然的方式去引导（用之不勤）。尊重事物原本的生长节奏，像母亲爱护孩子一样对待自己的事业与生活。这种“柔弱胜刚强”的力量，反而是成就大事的关键。
