# 第 16 章：知常曰明，不知常，妄作，凶

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
致虚极，守静笃（dǔ），万物并作，吾以观复。
夫物芸芸，各复归其根。归根曰静，是谓复命。
复命曰常，知常曰明，不知常，妄作，凶。
知常容，容乃公，公乃王（wàng），王（wàng）乃天，天乃道，道乃久，没（mò）身不殆。
```

## 解释
**致虚极，守静笃（dǔ）。**
尽力使心灵达到极度的虚空，彻底守住内心的极度清静。

**万物并作，吾以观复。**
万物都在蓬勃生长，我从中观察它们循环往复的规律。

**夫物芸芸，各复归其根。归根曰静，是谓复命。**
世间万物虽然形态多样、生长繁茂，但最终都会回归到它们的本根。回归本根就叫做“静”，这种“静”就是回归到了生命的本真状态。

**复命曰常，知常曰明，不知常，妄作，凶。**
回归本真就是万物运行的永恒规律，认识这种规律叫做“明”（明智）；如果不了解这种规律而轻举妄动，就会招致凶险。

**知常容，容乃公，公乃王（wàng），王（wàng）乃天，天乃道，道乃久，没（mò）身不殆。**
认识了规律就能做到宽容，宽容了就能做到公正，公正了就能做到周全，周全了才符合自然（天），符合自然就契合了“道”。契合了“道”就能长久，终身都不会遭遇危险。

## 分析
本章提出了“清静”和“归根”的核心思想。老子认为，万物的发展都有一个从生长到回归的循环过程，而“静”才是事物的本质和规律。只有当一个人内心达到极致的虚空和宁静时，才能观察到这种深层规律，并顺应它。

在快节奏的现代生活中，我们经常处于焦虑和盲目扩张的状态，总想“做加法”，却忘记了“归根”。这种违背自然规律的“妄作”，往往会导致身体透支、事业挫败或心理失衡。老子提醒我们，要在喧嚣中守住内心的宁静，在认清事物本质后再行动。

我们要学会“知常”。在职场中，这代表要理解行业周期、人际常情和客观规律，而不是逆势而为。在个人成长中，这代表要定期让自己“清零”（致虚极），抛弃成见和杂念，回归到最朴素的状态去思考。只有内心宽广、处事公正的人，才能走得更远，实现真正的长久。
