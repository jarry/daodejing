# 第 81 章：为而不争
> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
信言不美，美言不信；善者不辩，辩者不善；知者不博，博者不知。
圣人不积。既以为人己愈有，既以与人己愈多。
天之道，利而不害；圣人之道，为而不争。
```

## 解释
**信言不美，美言不信；善者不辩，辩者不善；知者不博，博者不知。**
真实可信的话听起来往往不漂亮，华丽好听的话往往不可信。善良的人不巧言善辩，巧辩的人往往并不真诚善良。真正得道的人不追求博杂（博而不精），卖弄学问的人并没有掌握道的真谛。

**圣人不积。**
圣人（有道的统治者）并不为自己积蓄和占有财富。

**既以为人己愈有，既以与人己愈多。**
圣人尽力帮助他人，自己反而感到更充实；圣人尽可能地给予他人，自己反而收获更多。

**天之道，利而不害；圣人之道，为而不争。**
上天的规律，是利于万物而不加妨害；圣人的准则，是努力有所作为却不为了私利去争夺。

## 分析
作为《道德经》的终篇，老子给出了全书的总结。他提醒我们，通往真理的道路往往是朴实的，而不是花哨的。无论是言语还是学问，返璞归真才是最高境界。

“圣人不积”揭示了财富与能量的流动定律。一个人如果只知道占有，最终会枯竭；而懂得“为人”与“与人”的人，其生命力会像泉水一样源源不断。利他，本质上是最高级的利己，因为它让个体与整体实现了共赢。

“为而不争”是全书的灵魂。老子并不主张消极躺平，而是鼓励“为”——去创造，去建设，去服务。但这种作为是不带私心的，不为了抢夺名利而破坏自然的和谐。带着这种心态做事，人就能从内耗中解脱，达到“无为而无不为”的至高境界。
