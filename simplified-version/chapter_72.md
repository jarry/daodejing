# 第 72 章：民不畏威，则大威至矣
> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
民不畏威，则大威至矣。
无狎（xiá）其所居，无厌其所生。
夫唯不厌，是以不厌。
是以圣人自知不自见（xiàn），自爱不自贵。
故去彼取此。
```

## 解释
**民不畏威，则大威至矣。**
当百姓不再畏惧统治者的高压政策和武力震慑时，真正大规模的反抗斗争就会来临了。

**无狎其所居，无厌其所生。**
不要逼迫人民不得安居，不要阻塞人民谋生的道路。让百姓能安稳生活、正常劳作是治理的根本。

**夫唯不厌，是以不厌。**
只有统治者不去压迫、厌弃人民，人民才不会厌恶和反抗统治者。

**是以圣人自知不自见，自爱不自贵。**
因此，有道的圣人有自知之明但不自我表现，有自爱之心但不自显高贵。

**故去彼取此。**
所以要去掉那些虚荣的彰显与显摆，保持内在的自知与自爱。

## 分析
老子在本章中严肃告诫掌权者：不可低估民众的力量。民生是国家的根基，如果百姓连最起码的生计都难以维持，所谓的“权威”也就失去了震慑力。当人们“不畏死”时，任何高压手段都会失效。

真正的自重不是靠高高在上的身份，而是靠“自知”与“自爱”。圣人之所以受人尊敬，是因为他把百姓看得很重，而把自己看得很轻。这种“不自贵”的姿态，反而赢得了民心的归顺。

在现代管理中，这同样适用。一个优秀的领导者不应通过打压员工来确立权威，而应通过保障团队利益、保持谦逊的态度来赢得尊重。去掉虚张声势，回归服务本质，才是长治久安之道。
