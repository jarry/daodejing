# 第 18 章：大道废，有仁义

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
大道废，有仁义；慧智出，有大伪；
六亲不和，有孝慈；国家昏乱，有忠臣。
```

## 解释
**大道废，有仁义；**
当宇宙根本规律（大道）被废弃、自然秩序被破坏时，才开始提倡所谓的“仁义”道德。

**慧智出，有大伪；**
当人们开始炫耀聪明、推崇智巧时，虚伪狡诈的风气也就随之出现了。

**六亲不和，有孝慈；**
当父子、兄弟、夫妇等亲属关系不和睦时，才需要刻意强调“孝顺”和“慈爱”。

**国家昏乱，有忠臣。**
当国家政局混乱、动荡不安时，才会有所谓的“忠臣”显现出来。

## 分析
老子用这一章揭示了“道德标杆”出现的深层背景。他认为，当社会处于最自然、最有道的理想状态时，仁义、孝慈、忠诚这些特质是像呼吸一样自然存在的，根本不需要专门提倡。一旦这些词被频繁挂在嘴边，往往说明社会的基础价值观已经崩塌。

这带给我们一种“反向思维”的洞察。在职场或家庭中，如果一个公司天天喊“忠诚”，可能说明员工流失严重；如果一个家庭天天强调“面子”和“礼节”，可能内部已经缺乏真诚的关爱。我们要警惕那些流于表面的形式主义，因为过度的修饰往往是为了掩盖本质的缺失。

我们应该追求“治未病”的状态，回归到最朴素的规则和感情中。不要等到关系破裂了才去修补，也不要等到危机出现了才找英雄。在日常生活中，保持诚信、守时、尽责等基本常识，让一切回归自然。当你不再需要用华丽的辞藻来证明自己的时候，你才真正契合了“道”。
