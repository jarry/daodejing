# 第 38 章：上德不德，是以有德

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
上德不德，是以有德；下德不失德，是以无德。
上德无为而无以为，下德为之而有以为。
上仁为之而无以为，上义为之而有以为，上礼为之而莫之应，则攘(rǎng)臂而扔之。
故失道而后德，失德而后仁，失仁而后义，失义而后礼。夫礼者，忠信之薄(bó)而乱之首。
前识者，道之华而愚之始。
是以大丈夫处其厚，不居其薄(bó)；处其实，不居其华。
故去彼取此。
```

## 解释
**上德不德，是以有德；下德不失德，是以无德。**
上等的“德”不表现为德的形式（顺其自然），所以它是真正有德的；下等的“德”处处想不失去德（刻意表现），所以它实际上没有达到德的境界。

**上德无为而无以为，下德为之而有以为。**
上德的人顺应自然且无心而为；下德的人刻意去做且心怀目的。

**上仁为之而无以为，上义为之而有以为，上礼为之而莫之应，则攘(rǎng)臂而扔之。**
上仁的人去做事但出于无心；上义的人去做事且出于有意。上礼的人去做事却没人响应，于是就撸起袖子强迫别人顺从。

**故失道而后德，失德而后仁，失仁而后义，失义而后礼。夫礼者，忠信之薄(bó)而乱之首。**
所以，丧失了“道”才强调“德”，丧失了“德”才强调“仁”，丧失了“仁”才强调“义”，丧失了“义”才强调“礼”。所谓“礼”，是忠信淡薄的产物，也是祸乱的根源。

**前识者，道之华而愚之始。是以大丈夫处其厚，不居其薄(bó)；处其实，不居其华。故去彼取此。**
所谓的“先见之明”，不过是“道”的虚华表象，也是愚昧的开端。因此，大丈夫立身处世应当选择敦厚，不选择轻薄；选择朴实，不选择虚华。应当舍弃表象而采取本质。

## 分析
本章是《道德经》中对社会价值观最深刻的层级划分。老子认为，最高的境界是“道”，那是与自然合一的。当人们开始强调“仁、义、礼”这些具体规范时，说明社会已经背离了最纯粹的本质。这种分级提醒我们：越是依赖外在的、形式化的规范，往往说明内在的、核心的信任已经缺失。

在职场文化中，如果一个公司开始过度强调繁琐的报销制度、严格的考勤礼仪或口号式的价值观，往往说明其团队的核心凝聚力和自驱力正在丧失。老子告诉我们，不要被这些“虚华”的表象所迷惑。一个真正强大的团队，靠的是内在的共识（道）和自然的担当（上德），而不是形式上的约束（礼）。

我们要追求“处其厚，居其实”的人生境界。不要做一个精于算计、追求表面光鲜的“聪明人”，而要做一个敦厚朴实、坚守本质的“大丈夫”。在处理人际关系和事业发展时，少一点套路（薄、华），多一点真诚和底蕴（厚、实）。守住了根本，你才不会在社会的浮躁中迷失自我。
