# 第 80 章：小国寡民
> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
小国寡民。
使有什佰（shí bǎi）之器而不用，使民重死而不远徙（xǐ）。
虽有舟车，无所乘之；虽有甲兵，无所陈之。
使民复结绳而用之。
甘其食，美其服，安其居，乐其俗。
邻国相望，鸡犬之声相闻，民至老死，不相往来。
```

## 解释
**小国寡民。**
理想的国家应该领土小一点，人口少一点。

**使有什佰之器而不用，使民重死而不远徙。**
即使有各种先进、高效的工具和器械，也不轻易使用；让百姓珍惜生命（重死），不需要为了生计或躲避战火而长途跋涉地迁徙。

**虽有舟车，无所乘之；虽有甲兵，无所陈之。**
虽然有船只和车辆，但并不一定要去乘坐（自给自足，不外求）；虽然有武器装备，却没有必要把它们陈列在阵前炫耀或打仗。

**使民复结绳而用之。**
让百姓回到远古结绳记事那种淳朴信实的时代，少一些尔虞我诈。

**甘其食，美其服，安其居，乐其俗。**
让百姓吃得香甜，穿得漂亮，住得安适，过得开心。这才是治理的终极目标。

**邻国相望，鸡犬之声相闻，民至老死，不相往来。**
邻国之间互相看得见，鸡鸣狗叫声互相听得到，但百姓一生生活安稳，彼此并不需要互相侵扰或争夺。

## 分析
本章是老子的“理想国”蓝图。很多人误解这是消极退步，其实老子是针对当时战火纷飞、大国争霸的惨状，提出的一种“反内卷”方案。他希望人们能摆脱外物的诱惑，回归到对生活本身（食、衣、居、俗）的享受中。

“小国寡民”强调的是一种适度与平衡。当一个组织或国家无限扩张时，管理成本和战争风险会激增。老子主张的“不用什佰之器”，本质上是反对“唯技术论”和“战争武器”，希望科技能向善，而不是成为奴役和杀戮的工具。

真正的幸福不在于“往来”有多频繁，而在于内心的“安”与“乐”。在现代社交过载的时代，学会“老死不相往来”式的独处，守住自己的那份淡定与满足，其实是一种极高的精神境界。
