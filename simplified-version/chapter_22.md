# 第 22 章：曲则全，枉则直

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
曲则全，枉则直，洼则盈，敝则新，少则得，多则惑。
是以圣人抱一，为天下式。
不自见（xiàn）故明，不自是故彰，不自伐故有功，不自矜故长。
夫唯不争，故天下莫能与之争。
古之所谓曲则全者，岂虚言哉！诚全而归之。
```

## 解释
**曲则全，枉则直；洼则盈，敝则新；少则得，多则惑。**
委曲反而能保全，弯曲反而能伸直；低洼反而能充盈，陈旧反而能更新；少取反而能获得，贪多反而会迷惑。

**是以圣人抱一，为天下式。**
所以，有智慧的人坚守“道”的原则（抱一），把它作为治理天下的范式。

**不自见（xiàn）故明，不自是故彰，不自伐故有功，不自矜故长。**
不自我表现，反而能看得分明；不自以为是，反而能名声彰显；不自我夸耀，反而能获得功勋；不自高自大，反而能长久领先。

**夫唯不争，故天下莫能与之争。**
正因为不与人争，所以天下没有人能与之争胜。

**古之所谓曲则全者，岂虚言哉！诚全而归之。**
古人所说的“委曲求全”，难道是空话吗？它是实实在在可以达到的境界。

## 分析
本章提出了著名的“不争之德”和辩证法。老子认为，事物的状态往往向相反的方向转化：低姿态反而能接纳更多，谦虚反而能获得更多关注。这种“以退为进”的思想，是一种极高明的处世智慧，它避开了尖锐的对抗，而通过顺应规律来达成目标。

在现代竞争中，我们习惯了凡事都要“争第一”、“显锋芒”、“证明自己”。但处处争强好胜往往会招致嫉妒、抵触和不必要的阻力。老子告诉我们，真正的强大不是通过压制别人获得的，而是通过自我完善和利他精神实现的。当你把心态放低（如洼、如曲），你反而成了能量的汇聚点。

我们要学会“抱一”，守住内心的平和。在团队中，不抢功、不自傲（不自伐、不自矜），反而会让大家心悦诚服地追随你。这种“不争”并不是消极放弃，而是一种战略上的格局。当你不再为小名小利纠结时，你就能专注于更有价值的事业，最终达成那种“莫能与之争”的至高境界。
