# 第 39 章：天得一以清，地得一以宁

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
昔之得一者：天得一以清，地得一以宁，神得一以灵，谷得一以盈，万物得一以生，侯王得一以为天下贞。
其致之。天无以清将恐裂，地无以宁将恐发，神无以灵将恐歇，谷无以盈将恐竭，万物无以生将恐灭，侯王无以贵高将恐蹶（jué）。
故贵以贱为本，高以下为基。
是以侯王自谓孤寡不穀（谷gǔ）。此非以贱为本邪（yé）？非乎？
故致数（shuò）舆（yù）无舆。不欲琭（lù）琭如玉，珞（luò）珞如石。
```

## 解释
**昔之得一者：天得一以清，地得一以宁，神得一以灵，谷得一以盈，万物得一以生，侯王得一以为天下贞。**
往昔得到“道”（一）而成就的例子有：天得到它而清明，地得到它而安宁，神灵得到它而显灵，河谷得到它而充盈，万物得到它而生长，君王得到它而成为天下的正统（大治）。

**天无以清将恐裂，地无以宁将恐发，神无以灵将恐歇，谷无以盈将恐竭，万物无以生将恐灭，侯王无以贵高将恐蹶（jué）。**
推而言之：天如果不清明恐怕会崩裂，地如果不宁静恐怕会震溃，神灵如果不显灵恐怕会消失，河谷如果不充盈恐怕会枯竭，万物如果不生长恐怕会灭绝，君王如果没有正道而只求高位恐怕会垮台。

**故贵以贱为本，高以下为基。是以侯王自谓孤寡不穀。此非以贱为本邪？非乎？**
所以，高贵以卑贱为根本，高大以下位为基础。因此君王们自称为“孤”、“寡人”、“不善”。这不正是以卑贱为根本吗？不是吗？

**故致数（shuò）舆（yù）无舆。不欲琭（lù）琭如玉，珞（luò）珞如石。**
所以，追求极度的赞誉反而没有赞誉。不要追求像美玉那样华丽，而应当像坚石那样朴实。

## 分析
本章论述了“道”（一）的统一性以及“贵贱相依”的底层逻辑。老子认为，宇宙间的万事万物，无论多么宏大（如天、地）还是多么高贵（如君王），其存在的根基都源于那个最基本、最质朴的“一”。更重要的是，他指出“贵”必须扎根于“贱”，“高”必须建立在“下”之上，一旦脱离了基础，就会面临毁灭。

这一章带给我们极其深刻的管理和生存启示。在职场中，如果你是一名管理者（高），千万不要忘记你的根基是基层员工和客户（下）。如果你自视甚高、脱离群众，就像“天无以清”一样，随时面临崩塌。君王自称“孤寡”，实际上是在提醒自己要保持低姿态，守住那个支撑自己的根本。

我们要学会“不欲琭琭如玉”。在生活中，不要去追求虚荣的标签、昂贵的装饰和过度的赞美（如玉）。这些东西不仅脆弱，还会让你脱离现实。相反，要学做那一颗“珞珞如石”的坚石，保持朴实、坚韧和接地气的状态。只有当你守住了那个最真实的“一”，守住了你的人格底线和底层基础，你的事业和人生才能真正长久。
