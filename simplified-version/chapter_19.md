# 第 19 章：见素抱朴，少私寡欲

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
绝圣弃智，民利百倍；
绝仁弃义，民复孝慈；
绝巧弃利，盗贼无有。
此三者，以为文不足，故令有所属，见（xiàn）素抱朴，少私寡欲。
```

## 解释
**绝圣弃智，民利百倍；**
抛弃掉所谓的“圣人智慧”和“智巧谋略”，老百姓反而能得到百倍的好处。

**绝仁弃义，民复孝慈；**
抛弃掉那些虚伪的“仁爱”和“道义”口号，老百姓反而能回归到天性中的孝顺和慈爱。

**绝巧弃利，盗贼无有。**
抛弃掉那些投机取巧和争权夺利的行为，世上也就不会有盗贼了。

**此三者，以为文不足，故令有所属，见（xiàn）素抱朴，少私寡欲。**
“圣智、仁义、巧利”这三者作为治国方略是不够的，还需要让人们的心灵有所归依：那就是展现质朴、保持淳朴（像未经染色的生丝和没雕琢的木头），减少私心，淡泊欲望。

## 分析
本章是老子对“返璞归真”的深刻呼吁。他认为人为的干预和复杂的说教往往会扭曲人性，产生虚伪和争斗。只有当统治者带头摒弃智巧和贪利，带头过简朴的生活，社会的道德水平才会自然回升。这是一种回归常识、回归本能的智慧。

在当今内卷严重的职场，我们习惯了各种“管理工具”、“沟通技巧”和“商业互吹”。每个人都戴着精致的面具，表现得极具智慧和职业精神，但内心却感到疲惫。老子告诉我们，过度的“包装”和“智巧”不仅累人，还可能导致信任的缺失。有时候，最真诚的朴实反而最具力量。

我们要学会在生活中做“减法”。“见素抱朴”不是让我们变笨，而是让我们在复杂的环境中守住那份单纯和底线。减少不必要的欲望和攀比（少私寡欲），你会发现生活变得简单明了。在管理或合作中，真诚待人、少玩心机，往往能获得意想不到的效率和长久的合作。
