# 第 77 章：天之道，其犹张弓与
> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
天之道，其犹张弓与？
高者抑下，下者举之，有余者损之，不足者补之。
天之道，损有余而补不足。
人之道，则不然，损不足以奉有余。
孰能有余以奉天下，唯有道者。
是以圣人为而不恃，功成而不处，其不欲见（xiàn）贤。
```

## 解释
**天之道，其犹张弓与？**
客观自然的运行规律，就好像拉弓射箭一般。老子借用拉弓来形容天道运行的平衡特点。

**高者抑下，下者举之，有余者损之，不足者补之。**
拉弓时，弦拉得高了就要压低一点，低了就再抬高一点；张得过满了就减少一点力气，拉得不足就再增补一点。只有不偏不倚，才能命中目标。

**天之道，损有余而补不足。**
上天的规律，是减损多余的来补给不足的，以此维持自然的平衡，使世界生生不息。

**人之道，则不然，损不足以奉有余。**
人类的法则却往往相反：弱肉强食，从本就不够的人那里抢夺，用来奉献给已经过剩的人。这源于人的贪婪自私，导致贫富差距悬殊。

**孰能有余以奉天下，唯有道者。**
谁能减损自己的富余来补给天下呢？只有那些有道的统治者可以做到。他们掌握权力，若能效法天道，就能实现社会的相对平衡与长治久安。

**是以圣人为而不恃，功成而不处，其不欲见贤。**
因此，得道的圣人有所作为而不自恃有功，有所成就而不居功自傲，他并不愿意彰显自己的贤能。

## 分析
本章老子提出了极其深刻的“平衡思想”。自然界之所以能长久运转，是因为它有一套自动调节机制：损有余而补不足。这种“天之道”本质上是公平与公正的体现，而“人之道”往往因为私欲而走向极端，造成强者恒强、弱者愈弱的失衡。

“损不足以奉有余”是人类社会很多悲剧的根源。老子告诫领导者要抑制私欲，主动去打破这种贪婪的循环。一个成熟的社会或组织，应该学会在高速发展中建立补偿机制，让利于民，让利于基层，才能避免系统因为过度倾斜而崩溃。

“为而不恃”是一种高级的职场和人生修养。当我们取得成就时，不应将其视为私产，而应看作是顺应趋势的结果。保持低调、不显摆，不仅能避免嫉妒，更能让自己始终处于“虚怀若谷”的状态，不断吸纳新的能量。
