# 第 73 章：勇于敢则杀，勇于不敢则活
> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
勇于敢则杀，勇于不敢则活。
此两者，或利或害。
天之所恶，孰知其故？
是以圣人犹难之。
天之道，不争而善胜，不应（或作不言）而善应，不召而自来，繟（chǎn）然而善谋。
天网恢恢，疏而不失。
```

## 解释
**勇于敢则杀，勇于不敢则活。**
勇敢而肆无忌惮、缺乏敬畏（勇于敢），往往会招致杀身之祸；勇敢而心存敬畏、有所顾忌（勇于不敢），反而能保全性命、化险为夷。

**此两者，或利或害。**
这两种“勇”的表现，一个有害，一个有利。

**天之所恶，孰知其故？**
上天所厌弃的事物，谁能知道其中的奥秘和原因呢？自然规律对万物是平等的。

**是以圣人犹难之。**
因此，即便是圣人，在处理事情时也总是小心翼翼、如履薄冰，把简单的事也当做困难来谨慎对待。

**天之道，不争而善胜，不应而善应，不召而自来，繟然而善谋。**
天道的规律是：不去争夺却能自然取胜，不用言语（强制命令）却能得到响应，无需召唤而万物自归，从容坦然却能周密谋划。

**天网恢恢，疏而不失。**
自然的法网广大无边，虽然看起来稀疏，却绝不会漏掉任何一个奖赏或惩罚。

## 分析
本章讨论的是“敬畏之心”。老子区分了两种勇气：一种是外露的、狂妄的“敢”；另一种是内在的、克制的“勇”。真正的强者不是无所畏惧，而是深知规律的严密，从而对自然保持一份谦卑。

“天网恢恢”传达了一种因果律。我们做任何事情都有相应的代价，只有符合客观规律，才能用最少的代价获得最好的结果。逆天而行、盲目冲动，即便一时得逞，最终也难逃自然规律的制约。

在快节奏的商业社会，这种“有所顾忌”的勇气尤为重要。它提醒我们在决策时不要盲目狂热，要学天道的“不争而善胜”。通过完善自我、顺应趋势，让成功自然而然地到来，而不是强求。
