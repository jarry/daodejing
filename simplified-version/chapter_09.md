# 第 9 章：持而盈之，不若其已

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
持而盈之，不若其已；
揣（zhuī）而锐之，不可长保。
金玉满堂，莫之能守；
富贵而骄，自遗其咎。
功遂身退，天之道也。
```

## 解释
**持而盈之，不若其已；揣而锐之，不可长保。**
给器皿不断加水使其溢满，不如适可而止；不断捶打铁器使其过于锋利，反而容易折断，难以长久保持。

**金玉满堂，莫之能守；富贵而骄，自遗其咎。**
即使家中堆满了金玉财宝，也是难以永久守住的；如果因为大富大贵就变得骄横跋扈，那是自寻烦恼，自招祸殃。

**功遂身退，天之道也。**
当事业成功、名利双收之时，应当适时退避或让位，这才是符合自然运行的客观规律。

## 分析
本章提出了“物极必反、盈满则亏”的危机防范逻辑。世间万物的发展皆有其限度，超越了这个限度，事物就会走向反面。老子通过“满溢之水”与“过锐之铁”等生动比喻，告诫我们要时刻保持警惕，懂得“度”的艺术，不要盲目追求极致的占有与扩张。

在现代职场与创业中，这种“功成身退”的智慧尤为深刻。许多人因为一时的成功而自我膨胀，在决策中变得独断专行（富贵而骄），结果导致企业的迅速崩塌。老子并不是让我们放弃进取，而是教我们学会“留白”，在巅峰时刻懂得急流勇退，给他人机会，也给自己留出转型的空间。这种智慧，能有效防止个人英雄主义导致的战略性失误。

这一章对现代人的启迪是：戒贪守约。无论在财富、权力还是名望的追求上，都要学会知止。不要把所有的光芒都汇聚在自己身上，而是要学会在成功后将利益与荣誉共享。保持谦卑收敛的心态，在巅峰时寻找下一个出口，这样的人才能避开“物极必反”的劫难，真正守住长远的价值。
