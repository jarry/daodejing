# 第 1 章：道可道，非常道

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
道可道，非常道；
名可名，非常名。
无，名天地之始，
有，名万物之母。
故常无欲，以观其妙；
常有欲，以观其徼。
此两者同出而异名，
同谓之玄，玄之又玄，众妙之门。
```

## 解释
**道可道，非常道；名可名，非常名。**
道是可以说的，但说出来的就不是那个道（原本的、恒久的）了。道的概念和形态也是可以描述定义的，但描述出来就不是那个概念了。

**无，名万物之始。有，名万物之母。（或无名，万物之始也。有名，万物之母也。）**
无，是万物的起始。有，是万物的母体。

**故常无欲，以观其妙；常有欲，以观其徼。**
因此，始终保持无欲无求的状态，才能体察到道之无名无形的奥妙；只有保持持续的探究欲望，才能发掘万事万物有形的边界和特征。

**此两者同出而异名。同谓之玄，玄之又玄，众妙之门。**
有和无，是两种不同的表述，它们都来自于道，是道的基本形态和不同的表述。道玄妙高深，就像一扇大门，通过它才能洞悉万物的运行规律。道是理解万事万物发展变化的根本入口。

## 分析
“道”是万物的起始和母体，它无名无形，难以用语言完全表达。“道”是可以说的，只不过说出来的“道”，就不是原来意义上的那个“道”了，因为言语本身具有局限性。有些东西只可意会，无法言明，一旦被说出来，就会偏离其原本的含义，这就是“不可说”。

世界万事万物都来源于“道”。就像爱因斯坦认为宇宙大爆炸的奇点创造了万物，老子则认为“道”是万物的根源。我们可以将“道”类比为一种未分化的整体状态，但它又不是一种具体的客观存在。“道”包含“无”和“有”：“无”是一切的起始，“有”则是万物的母体。

要想理解“道”，就必须去感悟和体会它，不能仅依赖语言和概念。要像一个观察者一样，保持“无欲无求”的状态，去感受那份空灵和微妙；也要带着探究的欲望，去把握事物的边界与特征。

现实世界中充满着名称与概念，人们往往容易被这些表象所束缚。因此，要想探究宇宙万物的真相和本质，我们必须在使用语言的同时，不被其所局限，找寻“道”最真实的内涵。

“道”是认识万事万物之门。一旦理解了“道”，那么就能洞悉万事万物的规律与发展变化。这种理解，并不是掌控万物或超然物外，更不是得道成仙，而是更少违背其内在规律，从而更清晰地认识事物的发展与变化。
