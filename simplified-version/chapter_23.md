# 第 23 章：希言自然

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
希言自然。
故飘风不终朝（zhāo），骤雨不终日。
孰为此者？天地。
天地尚不能久，而况于人乎？
故从事于道者，道者同于道，德者同于德，失者同于失。
同于道者，道亦乐得之；同于德者，德亦乐得之；同于失者，失亦乐得之。信不足焉，有不信焉。
```

## 解释
**希言自然。**
少发号施令，是符合自然规律的。

**故飘风不终朝（zhāo），骤雨不终日。孰为此者？天地。**
所以狂风刮不了一个早晨，暴雨下不了一整天。是谁在主导这些呢？是天地。

**天地尚不能久，而况于人乎？**
连天地制造的这种剧烈变动都不能持久，更何况是人的行为和政令呢？

**故从事于道者，道者同于道，德者同于德，失者同于失。**
所以，按“道”办事的人，就会与“道”同在；按“德”办事的人，就会与“德”同在；而背离道的人，就会与“失败”同在。

**同于道者，道亦乐得之；同于德者，德亦乐得之；同于失者，失亦乐得之。**
志同道合的人，道也乐于接纳他；有德的人，德也乐于亲近他；而那些甘愿堕落的人，失败和过失也会常伴他左右。

**信不足焉，有不信焉。**
管理者的诚信不足，部下或百姓自然就不会信任他。

## 分析
本章核心在于“持久性”和“信任”。老子用自然界中“飘风骤雨不能持久”的现象，告诫管理者不要朝令夕改或用力过猛。任何极端的、强制性的行为都是违反自然的，也必然无法长久。真正的力量往往蕴含在平淡、持续和稳定的状态中。

在工作中，我们经常见到那种“新官上任三把火”或“短期冲刺式的疯狂加班”。这种做法虽然短期见效，但往往会造成人心的疲惫和系统的崩溃。老子告诉我们，与其追求短期的爆发，不如追求长期的“同于道”。只有建立在信用和规律之上的努力，才是真正可持续发展的。

我们要培养“希言”的定力。不要频繁地制定新规则或大谈空洞的口号，而是要让行动符合规律，让诚信成为根基。当你选择与正确的人（有德者）、正确的事（道者）站在一起时，正向的能量就会自然汇聚。记住，平淡守信的人，往往比那些咋咋呼呼、用力过猛的人走得更稳、更远。
