# 第 28 章：知其雄，守其雌

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
知其雄，守其雌，为天下溪。
为天下溪，常德不离，复归于婴儿。
知其白，守其黑，为天下式。
为天下式，常德不忒（tè），复归于无极。
知其荣，守其辱，为天下谷。为天下谷，常德乃足，复归于朴。
朴散则为器，圣人用之则为官长（zhǎng）。故大制不割。
```

## 解释
**知其雄，守其雌，为天下溪。**
深知什么是雄强，却甘愿守在雌柔的位置，像天下的溪涧一样（汇集各种水流）。

**为天下溪，常德不离，复归于婴儿。**
甘愿做天下的溪涧，永恒的德行就不会离失，从而回归到婴儿般纯真质朴的状态。

**知其白，守其黑，为天下式。**
深知什么是光明、显赫，却甘愿处在暗昧、卑下的地位，作为天下的模范。

**为天下式，常德不忒（tè），复归于无极。**
作为天下的模范，德行就不会出现偏差，从而回归到无穷无尽的“道”的状态。

**知其荣，守其辱，为天下谷。**
深知什么是荣耀，却甘愿守住卑微和屈辱，像天下的山谷一样（深邃且包容）。

**为天下谷，常德乃足，复归于朴。**
甘愿做天下的山谷，永恒的德行才会充足，从而回归到纯朴自然的本原（朴）。

**朴散则为器，圣人用之则为官长（zhǎng）。故大制不割。**
这种“朴”散发开来就能成就万物（器）；圣人运用这种精神，就能成为众人的首领。所以完美的制度或宏大的治理是浑然一体、不被割裂的。

## 分析
本章探讨了平衡的艺术和回归本真的路径。老子提出了三组对立的概念：雄与雌、白与黑、荣与辱。他认为，虽然我们要了解强大的、光明的、荣耀的一面，但更要守住柔软的、暗淡的、卑微的一面。这种“守”并非懦弱，而是一种蓄势，让自己像溪流和山谷一样，具有极强的包容力和持久的德行。

在追求成功的道路上，我们往往只盯着“雄、白、荣”——权力、名声、光鲜亮丽。为了这些，我们把自己雕琢得面目全非。老子告诉我们，真正的领导者必须具备“谷”和“溪”的胸怀。只有当你能承受辱、能守住静、能处在低下位置时，你才能汇聚资源、赢得信任。这种“复归于朴”的过程，其实是自我进阶的过程。

我们要学会保持内心的“婴儿”状态，即那份最初的真诚和纯粹。在职场中，即便你已经功成名就，也要时刻提醒自己不要掉入“荣”和“雄”的陷阱。保持虚怀若谷的姿态，不割裂看待荣辱，让自己的心境回归到那种未被雕琢的质朴。只有这样，你才能承载大任，成为真正的“官长”。
