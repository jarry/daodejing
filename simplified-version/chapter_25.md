# 第 25 章：人法地，地法天，天法道，道法自然

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
有物混（hùn）成，先天地生。
寂兮寥兮，独立不改，周行而不殆，可以为天下母。
吾不知其名，字之曰道，强(qiǎng)为之名曰大。
大曰逝，逝曰远，远曰反。
故道大，天大，地大，王亦大。
域中有四大，而王居其一焉。
人法地，地法天，天法道，道法自然。
```

## 解释
**有物混（hùn）成，先天地生。**
有一个浑然一体的东西（指道），早在天地形成之前就已经存在了。

**寂兮寥兮，独立而不改，周行而不殆，可以为天下母。**
它是那样的寂静和空虚，不依靠任何外力而独立存在且永恒不变，循环运行永不停息，可以作为万物的根源和母亲。

**吾不知其名，字之曰道，强(qiǎng)为之名曰大。**
我不知道它的真实名称，给它取个字叫“道”，再勉强给它起个名叫“大”。

**大曰逝，逝曰远，远曰反。**
“大”意味着它无边无际、向外延伸；延伸得极远，远到极点后又会返回本原。

**故道大，天大，地大，王亦大。域中有四大，而王居其一焉。**
所以说，道大，天大，地大，人（王）也大。宇宙中有这四个伟大的存在，而人是其中之一。

**人法地，地法天，天法道，道法自然。**
人效法大地的坚实厚道，地效法上天的运行规律，天效法“道”的运行法则，而“道”效法的是它自身的自然状态（回归本本真）。

## 分析
本章是《道德经》中最具宇宙观的篇章之一。老子构建了一个宏大的宇宙等级序列，并指出了人类在其中的位置。他认为“道”是最高规律，具有独立性、普遍性和循环性的特点。而“道法自然”这一命题，揭示了宇宙间最高级的法则就是顺应事物原本的样子。

这章为我们提供了一种极高的“升维思考”视角。老子告诉我们，人类虽然伟大（域中四大之一），但必须对自然规律保持敬畏。我们的一切行为、制度和决策，都应该向上追溯，去学习大地的承载、上天的守信和道的无私。违背自然的“人造法则”往往是脆弱的，只有契合自然的规律才能长久。

我们要学会在生活中寻求“天人合一”的和谐。当我们在工作或管理中遇到瓶颈时，不妨问问自己：这个方案是否符合事物发展的自然本性？是否过于刻意或强求？效法大地，就要学会包容和务实；效法上天，就要学会规律和守恒。顺应自然不是不作为，而是更高级的“无为而作”，通过顺势而为来实现最高效的成功。
