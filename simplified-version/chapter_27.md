# 第 27 章：善行无辙迹，善言无瑕谪

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
善行无辙迹，善言无瑕谪(xiá zhé)，善数（shǔ）不用筹策，善闭无关楗（jiàn）而不可开，善闭无关楗而不可开，善结无绳约而不可解。
是以圣人常善救人，故无弃人；常善救物，故无弃物，是谓袭明。
故善人者，不善人之师；不善人者，善人之资。
不贵其师，不爱其资，虽智大迷，是谓要妙。
```

## 解释
**善行无辙迹，善言无瑕谪(xiá zhé)；**
善于行走的人，不留下车辙或脚印（指做事不露痕迹）；善于言谈的人，不会留下瑕疵或被人诟病的把柄。

**善数（shǔ）不用筹策；**
善于计算的人，不需要借助筹码工具也能了然于胸。

**善闭无关楗（jiàn）而不可开，善结无绳约而不可解。**
善于关闭的人，不用栓锁，别人也无法开启；善于捆缚的人，不用绳索，别人也无法解开。

**是以圣人常善救人，故无弃人；常善救物，故无弃物，是谓袭明。**
所以有智慧的人善于发现每个人的长处，因此没有不可用的人；善于发掘每样东西的价值，因此没有不可用的物。这叫作内藏的明智。

**故善人者，不善人之师；不善人者，善人之资。**
所以，优秀的人可以作为平庸者的老师，而平庸的人可以作为优秀者的借鉴。

**不贵其师，不爱其资，虽智大迷，是谓要妙。**
如果不尊重老师，不珍惜借鉴，即使自作聪明，其实也是个大糊涂。这就是“道”中最精深奥妙的道理。

## 分析
本章论述了“无痕管理”和“包容并蓄”的智慧。老子用一系列“善于”作为比喻，说明真正的高手做事顺应自然，不刻意、不张扬，却能达到完美的控制。同时，他提出了“无弃人、无弃物”的观念，强调管理者应具备慧眼，善于转化资源，将每个人都放在合适的位置上。

在职场中，我们常看到一些管理者迷信规章制度（关楗、绳约），企图通过严厉的管控来维持秩序。老子告诉我们，最高级的管理是“无形”的：通过建立共识、文化和信任，让人心悦诚服地团结在一起。此外，一个优秀的领导者不应抱怨团队成员平庸，而应学会“救人”，把所谓“平庸者”转化为团队的宝贵资源或反思的借鉴。

我们要培养“袭明”的智慧，学会从不同的人和事中汲取营养。不仅要向成功者（师）学习，也要从失败者（资）身上吸取教训。不要轻易否定任何人或任何资源，尝试去发现它们被掩盖的价值。当你能做到万物皆为我所用、处世不留痕迹时，你也就接近了“道”的境界。
