# 第 17 章：功成事遂，百姓皆谓我自然

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
太上，下知有之。
其次，亲而誉之。
其次，畏之。其次，侮之。
信不足焉，有不信焉。
悠兮其贵言。功成事遂，百姓皆谓我自然。
```

## 解释
**太上，下知有之。**
最好的统治者（领导者），部下或百姓仅仅知道有这么一个人存在（而感觉不到他在干预）。

**其次，亲而誉之。**
次一等的统治者，百姓亲近他并赞美他。

**其次，畏之。其次，侮之。**
再次一等的统治者，百姓畏惧他；最差的一等，百姓轻蔑并侮辱他。

**信不足焉，有不信焉。**
统治者的诚信不足，百姓自然就不会信任他。

**悠兮其贵言。功成事遂，百姓皆谓我自然。**
最好的统治者总是悠闲自在，不轻易发号施令。当大事告成、事业圆满的时候，老百姓都觉得：“我们本来就是这个样子的（这是我们自己努力的结果）。”

## 分析
本章论述了管理者的四个境界。最高境界是“无为而治”，即管理者顺应规律，润物无声，让团队成员在不知不觉中完成目标，并获得成就感。这要求管理者放下权力的傲慢，把舞台留给下属，实现“大功无痕”。

在现代企业管理中，很多管理者喜欢刷“存在感”，频繁开会、下指令、展示权威。这往往属于“畏之”或“亲而誉之”的范畴。老子推崇的“太上”境界，类似于现代推崇的“赋能型领导”或“仆人式领导”，不居功自傲，而是构建一个良好的系统，让系统自我运作。

我们要学习这种“功成不居”的胸怀。在团队协作中，真正的领导力不是压制或控制，而是成就他人。当你能做到“信足”且“贵言”（不乱说话、不乱指挥），团队的凝聚力和自主性就会达到最高。让每个人都觉得成功是源于自己的努力，这才是最高超的管理艺术。
