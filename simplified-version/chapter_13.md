# 第 13 章：宠辱若惊，贵大患若身

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
宠辱若惊，贵大患若身。
何谓宠辱若惊？宠为下，得之若惊，失之若惊，是谓宠辱若惊。
何谓贵大患若身？吾所以有大患者，为吾有身，及吾无身，吾有何患！
故贵以身为天下，若可寄天下；爱以身为天下，若可托天下。
```

## 解释
**宠辱若惊，贵大患若身。**
受到宠爱或遭受侮辱都感到心惊不安；像看重自己的身体一样去看重大的祸患。

**何谓宠辱若惊？宠为下，得之若惊，失之若惊，是谓宠辱若惊。**
什么叫宠辱若惊？恩宠相对于施予者来说是居于下位的，得到它会感到惊喜不安，失去它则会感到惊恐不安。这就叫宠辱若惊。

**何谓贵大患若身？吾所以有大患者，为吾有身，及吾无身，吾有何患！**
什么叫看重祸患像看重身体一样？我之所以有大祸患，是因为我有这个身体（自我）；如果我没有了这个身体（忘掉自我），我还能有什么祸患呢？

**故贵以身为天下，若可寄天下；爱以身为天下，若可托天下。**
所以，能够像珍惜自己生命一样去珍惜天下的人，才可以把天下的重担交付给他；能够像爱护自己身体一样去爱护天下的人，才可以把天下的重责托付给他。

## 分析
本章探讨了对待荣辱和自我的态度。老子认为，宠和辱其实是相连的，本质上都是由于对“自我”的过度执着。人们因为太在乎自己的名利、地位和身体，所以才会患得患失。只有当一个人能够放下私欲，达到“无身”的境界，才能真正摆脱荣辱的束缚，承载更大的责任。

在职场中，我们常因为领导的褒奖而欣喜若狂，又因为一点批评而垂头丧气，这正是“宠辱若惊”的写照。这种情绪的剧烈波动源于我们把个人的面子和利益看得太重。如果我们将注意力从“自我表现”转移到“事情本身”或“团队责任”上，就能更加淡定地面对职业生涯的起伏。

我们要培养一种“大任”意识。只有那些不计较个人得失，把集体利益或天下公义看得和自己生命一样重要的人，才具备真正的领导力。在日常生活中，尝试减少对自我的执念，学会以全局视角思考问题。当你不再被小我所局限时，你的心胸会变得豁达，也能在更高的层次上实现自我价值。
