# 第 14 章：视而不见，名曰夷

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
视而不见，名曰夷；听之不闻，名曰希；
搏之不得，名曰微。此三者不可致诘（jié），故混而为一。
其上不徼（jiǎo），其上不昧（mèi），绳（mǐn）绳兮不可名，复归于无物。
是谓无状之状，无物之象，是谓惚恍。
迎之不见其首，随之不见其后。
执古之道，以御今之有。能知古始，是谓道纪。
```

## 解释
**视而不见，名曰夷；听之不闻，名曰希；搏之不得，名曰微。**
看不见它的形状，叫做“夷”；听不到它的声音，叫做“希”；摸不着它的实体，叫做“微”。

**此三者不可致诘（jié），故混而为一。**
这三种特质无法推究到底，因为它们本来就是浑然一体的（指“道”）。

**其上不徼（jiǎo），其上不昧（mèi），绳（mǐn）绳兮不可名，复归于无物。**
它的上面并不显得明亮，下面也不显得阴暗。它连绵不断，无法给出确切的名称，最终还是回归到无物状态。

**是谓无状之状，无物之象，是谓惚恍。**
它没有形状却有状态，没有实物却有形象，这就叫作“惚恍”（似有似无）。

**迎之不见其首，随之不见其后。**
迎着它走，看不见它的头；跟着它走，看不见它的尾。

**执古之道，以御今之有。能知古始，是谓道纪。**
秉持着自古以来的规律（道），来驾驭现今存在的事物。能够认识宇宙最原始的开端，这就是“道”的纲领。

## 分析
本章老子试图通过描述“道”的不可感知性，来阐明“道”的深刻内涵。“道”是超脱于感官之外的，它不具备具体的色、声、形，但它又是真实存在的宇宙根本规律。这种规律是统一的、延绵不断的，它是万事万物背后的终极驱动力。

在复杂多变的世界中，我们往往容易被表象所迷惑。老子告诉我们要透过现象看本质。无论是商业逻辑、自然规律还是人际准则，最核心的道理往往是看不见、摸不着的，但它们却决定了事物的走向。如果我们只盯着那些显而易见的指标，就会在“惚恍”中迷失方向。

我们要学会“执古之道，以御今之有”。这意味着要不断学习和感悟那些恒久不变的底层逻辑（如诚实、勤奋、因果律等），并用这些逻辑来指导我们解决当下的问题。在决策时，不要被短期的利诱和噪音干扰，而要思考这件事是否符合事物的基本规律。把握住了“道”的纲领，才能在纷纷扰扰的世间保持清醒。
