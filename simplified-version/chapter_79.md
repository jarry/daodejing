# 第 79 章：天道无亲，常与善人
> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
和大怨，必有余怨，安可以为善？
是以圣人执左契，而不责于人。
有德司契，无德司彻。
天道无亲，常与善人。
```

## 解释
**和大怨，必有余怨，安可以为善？**
和解深重的仇怨，心中难免还会留下残余的积怨，这怎么能算是妥善、彻底的解决办法呢？

**是以圣人执左契，而不责于人。**
所以有道的圣人虽然持有借据（契约的左半边），却并不以此苛刻地索取债务或责备他人。

**有德司契，无德司彻。**
有德之人就像持有借据的圣人一样宽容公正，只按契约行事而不逼迫；无德之人就像管租税的官吏（彻）那样刻薄，只知道压榨盘剥。

**天道无亲，常与善人。**
天道运行规律对任何人都没有偏爱，但它却总是眷顾和帮助那些有德的善人。

## 分析
本章讨论的是“宽容”与“因果”。大怨之所以难和，是因为伤害已经造成。老子建议我们从源头上“以德报怨”，通过圣人那种“执契而不责”的宽厚，来消弭社会中的戾气。

“天道无亲”并不是说老天爷有感情，而是说“德”本身就是一种顺应自然的属性。一个行善的人，其行为逻辑是符合“道”的，因此他受到的阻力最小，得到的助力最大。这就是所谓的“常与善人”。

在商业或社交中，守信（司契）是基础，而宽厚则是更高阶的智慧。不去穷追猛打他人的过错，给他人留余地，其实是在为自己积累未来的转机。做一个善良的人，并不是为了回报，而是为了让自己的生命状态与天道共振。
