# 第 11 章：有之以为利，无之以为用

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
三十辐共一毂（gǔ），当其无，有车之用。
埏埴（shān zhí）以为器，当其无，有器之用。
凿户牖（yǒu）以为室，当其无，有室之用。
故有之以为利，无之以为用。
```

## 解释
**三十辐共一毂（gǔ），当其无，有车之用。**
古时候车轮由三十根辐条汇集到中心的一个轮毂上，正因为轮毂中间有空隙（安装车轴的地方），车轮才能转动，发挥车子的作用。

**埏埴（shān zhí）以为器，当其无，有器之用。**
揉和黏土制作器皿，正因为器皿中间有空虚的部分，才能盛放东西，发挥器皿的作用。

**凿户牖（yǒu）以为室，当其无，有室之用。**
开凿门窗建造房屋，正因为房屋中间有空虚的部分（门窗和室内空间），人才可以居住，发挥房屋的作用。

**故有之以为利，无之以为用。**
所以，“有”（实体）供人便利，而“无”（虚空）才发挥了真正的效用。

## 分析
这一章老子论证了“有”和“无”的辩证关系。我们通常只看到实体的“有”，却忽略了看不见的“无”。老子通过车轮、器皿、房屋这三个生动的例子说明，实体的存在只是提供了便利的基础，而真正的功能和价值往往是由其中的“虚空”部分实现的。没有“无”，“有”就无法发挥作用。

在现代职场和生活中，我们往往过度追求看得见的“有”——金钱、头衔、物质财富，却忽视了看不见的“无”——内心的空间、思维的留白、休息的时间。一个人的精力如果是满的，就无法产生新的创意；一个团队如果只有KPI和规则，没有包容和自由的空间，就无法真正高效运转。

我们应当学会重视“无”的价值。在忙碌的工作中给自己留出“复盘”和“思考”的空白；在处理人际关系时，给对方留出空间；在学习新知识前，先清空旧有的偏见。只有保持“不盈”的状态，才能让“有”发挥出最大的效用，实现真正的成长与进步。
