# 第 12 章：五色令人目盲，五音令人耳聋

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
五色令人目盲，五音令人耳聋，五味令人口爽；
驰骋畋（tián）猎令人心发狂，难得之货令人行妨。
是以圣人为腹不为目，故去彼取此。
```

## 解释
**五色令人目盲，五音令人耳聋，五味令人口爽；**
过分追求色彩的绚烂会让人视觉迟钝，过分追求乐声的嘈杂会让人听觉失灵，过分追求滋味的浓厚会让人味觉受伤。

**驰骋畋（tián）猎令人心发狂，难得之货令人行妨。**
沉溺于骑马打猎的快感会让人心境狂乱，追求稀有珍贵的财宝会让人行为不轨、甚至走向犯罪。

**是以圣人为腹不为目，故去彼取此。**
所以有智慧的人只求基本的饱腹和生存（追求内在的安定），而不去追求感官的奢靡享受（外在的浮华）。因此他们会舍弃外在的物欲追求，而选择内在的简朴生活。

## 分析
老子在本章深刻揭示了“物极必反”和“过犹不及”的道理。人们往往认为物质享受越多、感官刺激越强就越幸福，但老子指出，过度的感官享受反而会导致知觉的丧失和精神的癫狂。这是一种对物欲的警示，强调应保持适度的欲望。

在物质充裕的现代社会，我们面临着前所未有的信息轰炸和感官诱惑。短视频、网红美食、奢侈品，这些“五色五音”不断消耗着我们的注意力和精力。如果我们一味追求感官的即时满足，最终会导致内心的空虚、焦虑甚至是行为的失准，迷失了生活的本真意义。

我们应当学会“为腹不为目”，关注内心真实的需求，而不是被外在的标签和欲望所绑架。在生活中提倡简约主义，减少不必要的消费和社交，把精力回归到身体健康、内心平和以及真正有意义的事情上。知足常乐，在简单朴素的生活中找寻最持久的幸福。
