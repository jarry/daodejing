# 第 34 章：以其终不自为大，故能成其大

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
大道泛兮，其可左右。
万物恃之而生而不辞，功成不名有，衣养万物而不为主，常无欲，可名于小；
万物归焉而不为主，可名为大。
以其终不自为大，故能成其大。
```

## 解释
**大道泛兮，其可左右。**
伟大的“道”广泛漫流，无处不在，左右上下无所不及。

**万物恃之而生而不辞，功成不名有。**
万物依赖它而生存，它从不推辞；它成就了万物，却不占有功名。

**衣养万物而不为主，常无欲，可名于小；**
它滋养覆盖万物，却不自以为是万物的主宰，它永远没有私欲，看起来似乎很渺小。

**万物归焉而不为主，可名为大。**
万物都归附于它，它依然不自以为主宰，这又可以称之为伟大。

**以其终不自为大，故能成其大。**
正因为“道”始终不自以为伟大（不显摆、不傲慢），所以才真正成就了它的伟大。

## 分析
本章探讨了“伟大”的真正定义。老子认为，真正的伟大（大道）是具备包容性和利他性的。它像空气和水一样滋养万物，却从不索取报酬，也从不要求权力。这种“不自为大”的姿态，反而让它赢得了万物的归附。这是一种“低姿态的高权威”。

在职场中，很多领导者陷入了“自以为大”的误区：通过展示权力、压制下属、频繁居功来维护自己的地位。老子告诉我们，这种做法其实是虚弱的表现。真正的领袖应该像“道”一样，为团队创造环境、提供资源，在成功时把荣誉留给团队，在平时不干扰大家的创造力。这种无私，反而会换来团队最高的忠诚度和战斗力。

我们要学习这种“功成不名有”的胸怀。在生活中，多做一些“润物无声”的好事，不计较回报，不追求显赫。当你不再渴望成为关注的焦点，而是专注于为他人创造价值时，你的影响力反而会像“大道”一样广泛漫流。真正的尊严和成就，从来不是要出来的，而是自然长出来的。
