# 第 75 章：民之饥，以其上食税之多
> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
民之饥，以其上食税之多，是以饥。
民之难治，以其上之有为，是以难治。
民之轻死，以其上求生之厚，是以轻死。
夫唯无以生为者，是贤于贵生。
```

## 解释
**民之饥，以其上食税之多，是以饥。**
百姓之所以遭受饥荒，是因为统治者征收的税收太沉重，耗尽了民脂民膏，所以才导致饥荒。

**民之难治，以其上之有为，是以难治。**
百姓之所以难以治理，是因为统治者贪图功名、强作妄为，频繁干涉民生，所以才引起反抗和混乱。

**民之轻死，以其上求生之厚，是以轻死。**
百姓之所以冒死轻生（轻视死亡），是因为统治者生活过于奢侈、奉养过于丰厚，逼得老百姓没有活路，只能以死相搏。

**夫唯无以生为者，是贤于贵生。**
只有那些不为了自己的私欲而恣意妄为的人，才胜过那些过分看重自身物质生活（厚养生命）的人。

## 分析
本章直接指出了阶级矛盾的根源。老子认为，百姓的“难治”和“轻死”都是被逼出来的结果。统治者的奢靡与普通人的生存困境形成了鲜明对比，这种不公才是社会动荡的温床。

“求生之厚”反而导致“失生”。一个人如果过度追求物质享受，甚至不惜剥削他人，这种极端的“贵生”其实是在自掘坟墓。反而是那些淡泊名利、顺应自然的“无以生为”者，能获得长久的安宁。

在企业管理中，这也提醒我们关注分配的正义。如果管理者拿走了绝大部分利润而无视员工的生计，团队必然“难治”。真正高明的领导者，懂得“让利于人”，通过藏富于民来换取长治久安。
