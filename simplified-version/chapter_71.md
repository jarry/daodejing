# 第 71 章：知不知，上
> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
知不知，上；不知知，病。
夫唯病病，是以不病。
圣人不病，以其病病，是以不病。
```

## 解释
**知不知，上；**
知道自己还有不知道的地方，这是一种高明。也就是我们常说的“知人者智，自知者明”，在已经掌握知识时仍保持谦虚，不自以为是，这是一种极高的修养。

**不知知，病。**
不知道却自以为知道，这是一种毛病和缺点。

**夫唯病病，是以不病。**
只有把缺点真正当做缺点（警查自己），才能最终没有缺点。敢于承认错误，敢于面对自己的不足，而不是自欺欺人。

**圣人不病，以其病病，是以不病。**
圣人之所以没有缺点，是因为他能正视自己的缺点并加以改进，所以才没有缺点。关键在于是否敢于承认问题之所在。

## 分析
本章老子揭示了人性的一个重大弱点：不懂装懂。真正入道的人，即便已经洞悉真理，也会保持虚怀若谷的状态，知道学无止境；而平庸的人稍有感悟便夸夸其谈，这在老子看来是一种“病态”。

苏格拉底曾说“认识自己的无知就是最大的智慧”，这与老子的“知不知”异曲同工。一个人只有承认自己的不足，才有修正和进步的空间。这种实事求是的态度，是自我提升的基石。

对于管理者而言，最忌讳“不知知”。自以为无所不知，就会导致刚愎自用，听不进不同意见。保持谦虚谨慎，不断通过复盘来发现自己的“病”，才能在复杂多变的环境中少犯错误，实现真正的“不病”。
