# 第 31 章：兵者不祥之器

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
夫佳兵者，不祥之器。物或恶（wù）之，故有道者不处（chǔ）。
君子居则贵左，用兵则贵右。
兵者，不祥之器，非君子之器。
不得已而用之，恬淡为上，胜而不美。
而美之者，是乐(yào)杀人。夫乐(yào)杀人者，则不可以得志于天下矣。
吉事尚左，凶事尚右。
偏将军居左，上将军居右，言以丧（sāng）礼处之。
杀人之众，以哀悲泣之，战胜，以丧礼处之。
```

## 解释
**夫佳兵者，不祥之器。物或恶（wù）之，故有道者不处（chǔ）。**
锋利的兵器是不吉祥的工具，人们都厌恶它，所以懂得“道”的人不会依仗它。

**君子居则贵左，用兵则贵右。**
君子平时以左方为尊贵（左代表生机、阳气），而在用兵打仗时则以右方为尊贵（右代表肃杀、阴气）。

**兵者，不祥之器，非君子之器。不得已而用之，恬淡为上，胜而不美。**
兵器是不吉祥的工具，不是君子所使用的。只有在万不得已的情况下才使用它，而且应当以恬淡的心态对待。即使打了胜仗，也不要觉得这是美事。

**而美之者，是乐(yào)杀人。夫乐(yào)杀人者，则不可以得志于天下矣。**
如果觉得打胜仗是美事，那就是在以杀人为乐。凡是以杀人为乐的人，是不可能在天下实现远大志向的。

**吉事尚左，凶事尚右。偏将军居左，上将军居右，言以丧（sāng）礼处之。**
吉祥的事以左方为上，丧葬的事以右方为上。在军队中，地位较低的偏将军居左，地位较高的上将军居右，这说明行军打仗应当被视作举办丧礼。

**杀人之众，以哀悲泣之，战胜，以丧礼处之。**
由于战争导致了大量人口伤亡，应当怀着悲哀的心情去面对；即便取得了胜利，也应当用丧礼的仪式来处理。

## 分析
本章表达了强烈的反战思想和对生命的尊重。老子认为战争是不祥之举，即便获得胜利，也绝不值得庆贺和炫耀，而应当视作一场沉重的丧礼。这种观点在战火纷飞的春秋战国时期显得尤为可贵，它将“不争”和“慈”的理念具体应用到了军事与政治领域。

在职场和商业竞争中，我们常把“商场如战场”挂在嘴边，甚至推崇“狼性文化”和恶意竞争。老子告诉我们，任何带有破坏性的手段都是“不祥之器”，只有在“不得已”的情况下才能作为防御手段使用。如果你以战胜对手、让对方“出局”为乐，这种心态本身就是一种危险的信号，最终无法赢得真正的尊重和长足的发展。

我们要培养一种“悲悯”的竞争观。当我们在竞争中获胜时，不要狂妄自大，而要反思代价，尊重对手，甚至为竞争带来的损耗感到遗憾。保持“恬淡”的心态，把精力放在价值创造而非单纯的胜负上。只有那些爱护生命、追求和谐的领导者，才能真正“得志于天下”，建立长青的基业。
