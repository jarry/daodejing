# 第 20 章：我独异于人，而贵食母

> 《道德经》简单解说版，用简单朴素语言解说。

## 原文
```
绝学无忧。唯之与阿（ē），相去几何？
善之与恶，相去若何？人之所畏，不可不畏。
荒兮其未央哉！众人熙熙，如享太牢，如春登台。
我独泊兮其未兆，如婴儿之未孩。傫傫（lěi）兮若无所归。
众人皆有余，而我独若遗。
我愚人之心也哉！沌沌兮！俗人昭昭，我独昏昏；俗人察察，我独闷闷。
澹（dàn）兮其若海，飂（liù）兮若无止。
众人皆有以，而我独顽似鄙。我独异于人，而贵食(sì)母。
```

## 解释
**绝学无忧。唯之与阿（ē），相去几何？善之与恶，相去若何？**
抛弃那些烦琐的成见和学问（指智巧之学），就没有忧患了。恭敬的答应与傲慢的呵斥，相差有多少呢？善良与丑恶，又相差多少呢？（其实都在一念之间）

**人之所畏，不可不畏。荒兮其未央哉！**
大家所畏惧的，不能不有所畏惧。这个道理广阔无际，没有尽头。

**众人熙熙，如享太牢，如春登台。我独泊兮其未兆，如婴儿之未孩。傫傫（lěi）兮若无所归。**
众人都是那么兴高采烈，好像在参加盛大的祭祀宴席，又像是在春天登台远眺。唯独我淡泊平静，无动于衷，像个还不会笑的婴儿。我疲惫散漫啊，好像一个找不到归宿的浪子。

**众人皆有余，而我独若遗。我愚人之心也哉！沌沌兮！**
众人都显得绰绰有余（追求名利），而我却好像一无所有。我真是长了一颗愚人的心啊，混混沌沌的。

**俗人昭昭，我独昏昏；俗人察察，我独闷闷。**
世俗之人都表现得聪明伶俐，唯独我看起来昏昏沉沉；众人都精于观察算计，唯独我闷声不响。

**澹（dàn）兮其若海，飂（liù）兮若无止。众人皆有以，而我独顽似鄙。我独异于人，而贵食(sì)母。**
我宁静得像大海，运动起来像飘忽的风没有止尽。众人都很有作为、很有才干，唯独我显得愚笨且低劣。我唯独与人不同的，就是看重并顺应万物的根源——“道”（食母：吸吮母体的乳汁，指依循道）。

## 分析
本章是老子孤独而坚定的自我独白。他用一系列强烈的对比，展现了“得道者”与世俗众人的巨大反差。当世人都在追求名利、表现聪明、享受享乐时，老子却甘愿保持一种“婴儿”般的质朴和“愚人”般的沉默。这并非消极，而是一种清醒的独立，不随波逐流。

在现实社会中，我们常常感到社交焦虑，害怕自己“不够聪明”、“不够成功”或“跟不上潮流”。为了迎合众人的审美和标准，我们把自己装扮得“昭昭”与“察察”，结果却深感疲惫。老子告诉我们，要守住自己的内心，即便看起来与世界格格不入，只要你是在遵循事物本质规律（食母），这种“愚”其实是大智慧。

我们要学会保持一份内心的宁静（澹兮其若海）。在职场竞争中，不一定要参与那些无意义的尔虞我诈和虚伪社交。回归质朴，把精力放在那些长期的、根本的规律上。当你不再被外在的评价（唯与阿、善与恶）所左右时，你就拥有了最坚定的力量。做一个有思想的“异类”，往往比做一个随声附和的平庸者更有价值。
